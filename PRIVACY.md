## 📋 Overview

This Privacy Policy describes how **CASPER TECH KENYA** (operated by **TRABY CASPER**) handles information in connection with the **Casper Free Proxies** project — including this GitHub repository and the REST API at `proxies.xcasper.space`.

We take your privacy seriously. This project is designed to collect as little data as possible.

---

## 📄 1. Information We Collect

### This GitHub Repository
When you visit, star, fork, or interact with this repository, **GitHub** collects information as described in [GitHub's Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement). CASPER TECH KENYA does not independently collect data from GitHub visitors.

### The REST API (`proxies.xcasper.space`)
When you make a request to our proxy API, the following information may be processed:

| Data | Purpose | Retained |
|------|---------|---------|
| IP address | Server-side request routing via Cloudflare | Not stored by us |
| Request path & method | Serving the correct response | Not stored by us |
| User-Agent header | General server logging | Not stored by us |
| Timestamp | Request processing | Not stored by us |

**We do not:**
- Log or store individual request details beyond standard transient server memory
- Set cookies or tracking pixels
- Use analytics on the API endpoints
- Sell, share, or monetize any request data

## 🌐 2. Third-Party Services

### Cloudflare
Our API domain `proxies.xcasper.space` is proxied through **Cloudflare**. Cloudflare processes traffic on our behalf and may collect metadata such as IP addresses and request timestamps for security and performance purposes. This is governed by [Cloudflare's Privacy Policy](https://www.cloudflare.com/privacypolicy/).

### GitHub
This repository is hosted on **GitHub** (owned by Microsoft). All repository interactions are subject to [GitHub's Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement).

### Third-Party Proxies
The proxy addresses in this repository are operated by **unknown third parties**. If you route traffic through any listed proxy, that proxy's operator may collect, log, or monitor your traffic. **CASPER TECH KENYA has no visibility into, or control over, what data third-party proxy operators collect.** We strongly advise you not to transmit sensitive or personal data through any proxy you do not personally control.

## 🔒 3. Data Security

We implement reasonable technical measures to protect the integrity of this project:

- API infrastructure is protected by Cloudflare's DDoS protection and firewall
- No user databases or accounts are maintained
- No passwords, credentials, or personal data are collected or stored

Despite these measures, no system is completely secure. We cannot guarantee the security of data transmitted through third-party proxy servers listed in this repository.

## 👤 4. Your Rights

As this project does not collect personal data, there is no personal data to access, correct, or delete. If you believe we have inadvertently collected information about you, please contact us immediately.

If you are located in the **European Union**, you may have rights under the **General Data Protection Regulation (GDPR)**. If you are in **California**, you may have rights under the **California Consumer Privacy Act (CCPA)**. Please contact us to exercise any applicable rights.
